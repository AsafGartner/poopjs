Reactive UI components

Write a function that updates the view and it will automatically and efficiently update the resulting HTML

<span class="number"></span>
<button>Click me!</button>
<script>
    var count = 0;
    var numberContainer = document.querySelector(".section.two .interactive .number");
    var button = document.querySelector(".section.two .interactive button");
    numberContainer.textContent = count;
    button.addEventListener("click", function() {
        count++;
        numberContainer.textContent = count;
    });
</script>

Silky smooth

Fast enough to power animations at 60FPS

The above animation is 128 <div> tags. No SVG, no CSS transitions/animations. It's all powered by Poop which updates just the right stuff every frame.
